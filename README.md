# Getting started
**!!! Important !!!** : `example_Swarm_Coordination.py` is NOT intended to be executed _outside the examples folder_ of [ZhiweiWei-NAMI/AeroAgentSim](https://github.com/ZhiweiWei-NAMI/AeroAgentSim).

Follow this steps to run the simulation:

1. Clone the project [ZhiweiWei-NAMI/AeroAgentSim](https://github.com/ZhiweiWei-NAMI/AeroAgentSim)
2. Follow the [AeroAgentSim installation guide](https://github.com/ZhiweiWei-NAMI/AeroAgentSim?tab=readme-ov-file#installation)
3. Copy the `example_Swarm_Coordination.py` into the path `AeroAgentSim/src/aeroagentsim/examples`
4. Be sure the virtual environment is activated
5. From `$ ~/path/to/AeroAgentSim` run `python src/aeroagentsim/examples/example_Swarm_Coordination.py`

# Simulation stats
The folder `$ ~/path/to/AeroAgentSim/simulation_stats` contains the data related to each simulation executed running the step 5.
